# Kutlwano Take – Personal Portfolio Website

Live site: https://kutlwano-portfolio.vercel.app

My official personal portfolio showcasing who I am, my journey as a self-taught full-stack & cloud developer, and all my latest projects (and future) projects.

## Tech Stack
- HTML5
- CSS3 (with modern Flexbox & Grid)
- Vanilla JavaScript
- Responsive design (mobile-first)
- Deployed on Vercel

## Features
- Fully responsive layout (desktop, tablet, mobile)
- Smooth scrolling navigation
- Projects section with live demo & source links
- Clean, minimal design focused on speed and readability
- Dark/light mode toggle (coming soon)

## Projects Showcased
- AI Resume Analyzer (Appsmith + Google Gemini)
- AI Portfolio Assistant (AWS Elastic Beanstalk + CodePipeline)
- NYC 311 Data Dashboard (Python + Streamlit + AWS)
- Caption Craft – AI Social Media Caption Generator (React + Gemini API)
- Multi-Cloud Deployment Demo (Docker + Kubernetes + Terraform)
