Caption Craft – Secure Generative AI Caption Generator (Full AWS Deployment)

Built and deployed a full-stack AI-powered social media caption generator. Secured the Gemini API key by implementing a serverless proxy with AWS Lambda Function URL (zero client-side exposure). Backend runs on Elastic Beanstalk (Node.js + Express), frontend is pure HTML/CSS/JS served directly from Beanstalk, and all user interactions are persistently stored in Amazon RDS MySQL. Overcame real-world CORS, JSON parsing, and networking challenges to deliver a production-ready, secure application in the af-south-1 (Cape Town) region.


Tech: AWS Elastic Beanstalk, AWS Lambda (Function URL), RDS MySQL, IAM, Security Groups, Node.js, Express, vanilla JavaScript, HTML/CSS, Google Gemini API, Git

NYC 311 Social Services – Full Data Engineering Pipeline

Built a complete end-to-end data engineering solution using the official NYC 311 Service Requests dataset (800+ rows). Designed a normalized PostgreSQL schema on AWS RDS, wrote advanced SQL with JOINs, CTEs, window functions and aggregations to uncover social-service trends across boroughs. Engineered a robust Python ETL pipeline (pandas + sqlalchemy + boto3) that extracts raw CSV, cleans and validates data, applies business logic, loads into RDS PostgreSQL and archives to S3. Deployed an interactive Streamlit dashboard for real-time filtering and visualization of complaint volume, response times, and agency performance.

Tech: Python (pandas, sqlalchemy, boto3), PostgreSQL (complex SQL, CTEs, JOINs), AWS RDS, AWS S3, Streamlit, ETL Design, Data Cleaning & Validation, Git
